Device

A CANopen device.

This class represents a single addressable device (or node) on the bus and provides methods for manipulating the object dictionary.

Device exposes targeted protocols SDO, PDO, and Hearbeat.

new Device(channel: RawChannel, deviceId: number, edsPath: (string | null), heartbeat: boolean)

Extends EventEmitter

Parameters
channel (RawChannel) socketcan RawChannel object.
deviceId (number) device identifier.
edsPath ((string | null) = null) path to the device's electronic data sheet.
heartbeat (boolean = false) enable heartbeat production.
